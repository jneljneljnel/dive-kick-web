#install
--------

NODE:
- switch to node v4.1.1 or above.
- nvm install v4.1.1
- nvm use 4.1.1


Install some globals.
- npm i nw-builder -g
- npm i grunt-cli -g
- npm i gulp -g

Go to root:
- npm i
- bower i

How to build nwjs
- run `gulp build`
- cd into build folder
- run `npm i`
- run `grunt nwjs`
